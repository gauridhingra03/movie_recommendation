# Movie Recommendation System

A hybrid movie recommendation system using TF-IDF,
genre similarity, and cast similarity.

## Dataset
TMDB 5000 Movie Dataset  
https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## How to Run
1. Clone or download the repo
2. Ensure dataset files are in `dataset/`
3. Open `movie_recommender.ipynb`
4. Run all cells
